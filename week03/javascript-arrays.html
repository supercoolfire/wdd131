    Notes (Responsive Images) Overview

    The work of responsive image designis to support smaller page weights and deliver content based on screen size. You have already worked on basic image optimization (format, sizing, and quality). You also need to consider how images appear on different screen sizes. This activity's objective is to reduce page weight while maintaining clarity and functionality across screen sizes and viewports (the visible area of a web page).

    Notes (JavaScript Arrays) Overview

    An array is a data structure that stores multiple values in a single variable instead of declaring separate variables for each value.

    Optional Resources

    Reference JavaScript Arrays Reference– MDN

    This reference provides a complete list of array methods; several are used extensively in this course.

    Master JavaScript Array Methods Like a Pro

    1. push():Adds one or more elements to the end of an array and returns the new length. 
      
const planets = ['Mercury', 'Venus', 'Earth'];
planets.push('Mars');
console.log(planets); // Output: ["Mercury", "Venus", "Earth", "Mars"]
    2. pop():Removes the last element from an array and returns it. 
      
const fruits = ['apple', 'banana', 'cherry'];
const removedFruit = fruits.pop();
console.log(removedFruit); // Output: "cherry"
    3. shift():Removes the first element from an array and returns it. 
      
const cars = ['BMW', 'Mercedes', 'Audi'];
const shiftedCar = cars.shift();
console.log(shiftedCar); // Output: "BMW"
    4. unshift():Adds one or more elements to the beginning of an array and returns the new length. 
      
const animals = ['lion', 'tiger', 'leopard'];
const newLength = animals.unshift('cheetah', 'panther');
console.log(newLength); // Output: 5
    5. concat():Merges two or more arrays and returns the new array. 
      
const fruits1 = ['apple', 'banana'];
const fruits2 = ['cherry', 'orange'];
const allFruits = fruits1.concat(fruits2);
console.log(allFruits); // Output: ["apple", "banana", "cherry", "orange"]
    6. slice():Extracts a section of an array and returns a new array. 
      
const pizzaToppings = ['pepperoni', 'mushrooms', 'onions', 'sausage', 'green peppers'];
const vegetarianSlice = pizzaToppings.slice(1, 4);
console.log(vegetarianSlice); // Output: ["mushrooms", "onions", "sausage"]
    7. splice():Removes or adds elements from an array and returns the removed elements. 
      
const fish = ['trout', 'salmon', 'bass', 'pike'];
const removedFish = fish.splice(2, 1, 'catfish', 'perch');
console.log(removedFish); // Output: ["bass"]
console.log(fish); // Output: ["trout", "salmon", "catfish", "perch", "pike"]
    8. lastIndexOf():Returns the last index at which a given element can be found in the array. 
      
const fruits = ['apple', 'banana', 'cherry', 'banana'];
const lastBananaIndex = fruits.lastIndexOf('banana');
console.log(lastBananaIndex); // Output: 3
    9. join():Joins all elements of an array into a string. 
      
const colors = ['red', 'green', 'blue'];
const colorString = colors.join(',');
console.log(colorString); // Output: "red,green,blue"
    10. indexOf():Returns the first index at which a given element can be found in the array. 
      
const fruits = ['apple', 'banana', 'cherry', 'banana'];
const bananaIndex = fruits.indexOf('banana');
console.log(bananaIndex); // Output: 1
    11. reverse():Reverses the order of the elements in an array. 
      
const ingredients = ['flour', 'sugar', 'eggs', 'butter'];
const recipe = ingredients.join(', ');
console.log(recipe); // Output: "flour, sugar, eggs, butter"
